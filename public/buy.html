Buy an invite code

Pick a package and pay with Bitcoin (on-chain or Lightning). When the payment confirms you get a one-time invite code that activates anonymous access tokens on any device. The payment and your browsing are cryptographically unlinkable — the server can know a code was bought, never what it is used for.

Loading packages…

After payment you are sent to a claim page that reveals your code. Save that claim link (we also keep a copy in this browser) — it is the only way to retrieve the code. No email, no account.
